What you will learn

Botpress is the leading open-source platform for building and managing voice and chat bots. The easiest way to explain what Botpress does is to say that “Botpress is to bots what Wordpress is to websites”.

At the end of this guide, you will know:

  • What Botpress is and how it compares to the other frameworks
  • How to setup Botpress on your local machine
  • How to create and customize a bot

If you are already familiar with Botpress, the API Reference instead.
